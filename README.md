## About <a id="about"></a>

holiYo is an e-commerce web application built on the holo_cards codebase, designed specifically for buying and playing with Charizard trading cards. Users can browse our curated collection, favourite cards, build their deck, and purchase securely. Leveraging modern web technologies, we deliver a smooth, engaging experience for card enthusiasts.
